# Assumptions

Assumptions are things we are temporarily treating as true so implementation can proceed.

Every assumption should eventually be validated, revised, or discarded.

## Current Assumptions

### A1. The engine can remain domain-independent

We assume the core objects and learning mechanics can be implemented without referencing any domain-specific concept.

Status: Active

### A2. Model revision can be represented explicitly

We assume a Model can revise through Evidence-linked Revisions rather than being regenerated without history.

Status: Active

### A3. Public Markets is a suitable first domain

We assume Public Markets provides enough observable events and feedback density to validate the architecture quickly.

Status: Active
